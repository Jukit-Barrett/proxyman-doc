# 多个过滤器

# #1.什么事？

从构建2.22.0 +，您可以一次组合许多过滤器。

* 使用复杂的过滤条件轻松快速地过滤请求/响应
* 坚持你的最后一个过滤器配置
* 方便的快捷方式
* 各种过滤器: URL，请求标头，响应标头，正文，状态代码，方法，颜色和注释
* 与和/或所有过滤器结合使用

![多重过滤器](images/ExYFSpHUcAYVbKp_b30aadc8.jpeg)

### 2.快捷方式

| 快捷方式 | 描述 |
| -------- | -------------------------------------------- |
| ⌘ f | 显示滤镜并聚焦于最后一个 |
| ⌘ n | 克隆新过滤器 |
| 融合 ⌘ n | 移除电流滤波器 |
| ⌘ ↑ | 关注上一个过滤器 |
| ⌘ ↓ | 关注下一个过滤器 |
| ESC | 关闭过滤器 |
