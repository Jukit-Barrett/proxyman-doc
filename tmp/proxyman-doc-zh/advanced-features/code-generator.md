# 代码生成器

# #1.什么事？

代码生成器是一个有用的工具，可帮助您自动生成特定代码语言的HTTP/HTTPS请求。

下表将列出所有支持的语言:

* 卷曲
* HTTPie
* 哈尔
* 邮递员收藏2
* Axios
* Go
* Java HTTPClient
* Javascript jQuery
* 节点HTTP
* 节点提取
* Python请求
* Objective-C NSURLSession
* Swift Alamofire 5
* 斯威夫特·莫亚
* Swift URLSession

![生成Swift Alamofire 5代码] (图片/code-generator_8a9d0ffb.png)

# #2.如何打开代码生成器选项卡？

1.在请求面板选项卡上-> 单击 “” 按钮
2.新窗口出现-> 点击 “代码生成器” 复选框

* 新选项卡将添加到请求面板-> 选择任何语言来显示代码
