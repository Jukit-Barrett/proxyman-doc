# 重复

### 1.什么事？

重复工具是一个有用的工具，可以快速使用相同的HTTP标头和正文发出新的HTTP请求，以便在不更改浏览器或客户端的情况下查看新的响应。

### 2.这是干什么用的？

* 快速发出HTTP/HTTPS请求，以测试来自服务器的最新响应。
* 对于开发人员来说，在不手动重复请求的情况下测试服务器的更改非常有用。

{% 提示样式 = "info" %}
重复工具仅支持HTTP/HTTPS请求。对WS/WSS流量的重复操作可能失败。
{% endhint %}

{% 提示样式 = "info" %}
如果重复请求与任何规则匹配，则断点、映射本地和脚本工具可以修改重复请求。
{% endhint %}

### 3.如何使用

* 右键单击单个或多个请求-> ** 重复 **

![快速重复当前请求] (图像/Screen_Shot_2022-06-23_at_14_49_23_18391649.jpg)

{% 提示样式 = "info" %}
⌘ 重复当前选择的请求。
{% endhint %}

### 4.设置

* ** 请求超时 **: 在设置-> 工具选项卡-> 请求超时中: 定义请求将超时的一秒钟。使用0禁用它。在Proxyman 4.13.0或更高版本上可用
