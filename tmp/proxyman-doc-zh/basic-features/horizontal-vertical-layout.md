# 水平/垂直/窗口布局

可以更改请求和响应面板的UI布局，以优化您的空间来呈现内容。

* ** 可拆卸窗口 **: 将请求和响应带到分离的窗口
* ** 垂直 **: 请求和响应面板处于垂直模式
* ** 水平 **: 请求和响应面板处于水平模式

![](images/02_56e68ced.png)

| 快捷方式 | 描述 |
| -------------------- | ----------------- |
| 起 ** ⌘ 向上箭头 ** | 可拆卸窗口 |
| ⌘ ** 左箭头 ** | 切换左面板 |
| ⌘ ** 右箭头 ** | 垂直模式 |
| ⌘ ** 向下箭头 ** | 水平模式 |
