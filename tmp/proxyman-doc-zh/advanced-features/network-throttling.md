# 网络条件

### 1.什么事？

网络条件 (网络限制) v1允许开发人员模拟不利的网络环境。Proxyman提供各种预设配置文件:

| 配置文件 | 描述 |
| -------------------- | ---------------------------------------- |
| 100% 失去连接 | 100% 失去所有连接 |
| 网络非常差 | 下载/上传: <1mbps |
| 慢速网络 | 下载/上传: 5-10mbps |
| 中等网络 | 下载/上传: 10-20mbps |
| 2g (EDGE) | 下载: <240kbps，上传 <200kbps |
| 3g | 下载: <780kbps，上传 <300kbps |
| 4g (LTE) | 下载: 30-50mbps，上传5-10mbps |
| Wi-Fi | 下载: 25-40mbps，上传15-30mbps |
| Wi-Fi 802。11ac | 下载: 150-250mbps，上传70-100mbps |

### 2.受益？

* 各种预设配置文件: 3g，4g，Wifi，坏/中等网络等。
* 帮助开发人员模拟各种网络条件 (下载/上传带宽，丢包率，延迟)。
* 更容易在特定的网络条件下测试您的应用程序。
* 申请 ** 系统范围 ** 或 ** 特定域名 **。

{% 提示样式 = "info" %}
网络条件v1不允许您自定义配置文件。我们将在v2中实现它。
{% endhint %}

{% 提示样式 = "info" %}
为了更好地模拟现实生活，下载/上传带宽不是固定的，它可能在给定范围内随机化。
{% endhint %}

### 3.如何使用它？

您可以通过导航到工具-> 网络条件 (CMD + SHIFT + J) 或从右菜单上下文访问来访问该功能。
