# 绕过代理列表

# #1.绕过代理服务器列表

绕过代理Lis可帮助开发人员:

*✅定义 ** 从不 ** 转到Proxyman代理服务器的域列表
* 忽略一些噪声交通
* 代理到Proxyman应用程序时，避免由于SSL钉扎SSL错误。

# #2.如何使用它？

2.在工具菜单-> SSL代理列表-> 旁路代理列表中打开旁路代理列表
3.输入您的域名列表 (用逗号分隔，支持简单通配符)

# #3.它是如何工作的？

1.一旦启动Proxyman应用程序，Proxyman将使用Proxyman列表覆盖您的系统绕过代理列表。您可以在系统设置中找到系统设置-> Wi-Fi -> 您的Wifi热点-> 详细信息...-> 代理-> 绕过代理列表
2.当应用程序关闭-> Proxyman将恢复到原来的设置。
