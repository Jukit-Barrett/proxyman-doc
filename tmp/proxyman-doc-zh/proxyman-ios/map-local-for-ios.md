# Map Local for iOS

# # 这是什么？

地图本地是一个工具，帮助iOS开发人员:

* 映射URL与任何预定义的响应。它允许您使用新的响应快速测试，而无需等待后端服务器的真实端点。
* 您可以修改状态代码，标题和正文。
