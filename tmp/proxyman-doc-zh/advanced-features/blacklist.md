# 阻止列表

# #1.什么事？

当您希望在调试会话期间 ** 阻止或隐藏 ** 某些域时，阻止列表工具非常有用。

例如，您可以在以下情况下使用阻止列表:

* 阻止来自特定域的所有广告请求
* 阻止淹没工作空间的所有分析请求
* 阻止所有恼人的ping请求从您的应用程序，以减少出现在Proxyman的请求数量
* 从您的网站隐藏分析流量而不阻止它。

阻止列表中的所有阻止域将 ** 删除连接。**

# #2.阻止操作

阻止列表支持一些可以满足您需求的阻止操作:

<table><thead><tr><th width = "238.87890625"> 阻止操作 </th><th> 描述 </th></tr></thead><tbody><tr><td> 阻止和隐藏请求 </td><td> 匹配请求被阻止并且不显示在应用程序上。</td></tr><td> 阻止和显示 </td><td> 匹配的请求被阻止，但是在应用程序上显示被阻止的请求。</td></tr><td> 隐藏，但不阻止 </td><td> 只是隐藏匹配的请求而不阻止它们。如果您隐藏烦人的请求但不阻止它们，这很有用。</td></tr></tbody></table>

![创建阻止/隐藏规则] (图像/Screen_Shot_2022-01-10_at_14.43.38_c892a445.png)

# #3.如何使用它？

* ** 工具 ** 菜单-> ** 阻止列表 **
* 右键单击请求或域-> ** 工具 ** -> ** 阻止列表... **

{% 提示样式 = "成功" %}
** ⌘ ⌘ \[** 到快速打开阻止列表窗口
{% endhint %}

# #4.阻止来自给定应用程序的所有流量

* 在Proxyman macOS 5.24.0或更高版本上，您可以选择一个应用程序并阻止所有流量
