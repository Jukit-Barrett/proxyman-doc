# 复制为

Proxyman从右键单击菜单上下文提供各种复制功能:

* ** 复制URL **: 复制所选流的URL。
* Copy cURL * *: 生成cURL命令，包含header、cookies、query和body。
* ** 复制单元格值 **: 复制右键单元格的字符串值
* ** 复制为 **: cookie，标头，请求和响应的正文。
