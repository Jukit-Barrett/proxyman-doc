# 多个选项卡

Proxyman支持多个选项卡，以提高您的工作效率。

### 键盘

所有键盘快捷键与Safari的快捷键相同。

| 快捷方式 | 操作 |
| ----------------------------------------------- | ------------------------------------------------------- |
| <p></p><strong>⌘ + T</strong></p> | 打开新选项卡 |
| <p></p><strong>⌘ + 1，2，3</strong></p> | <p></p><p> 快速导航到第一，第二，第三选项卡。</p> |
| <p></p><strong>⌘ + + [or ]</strong> </p> | <p></p><p> 快速导航左侧或右侧选项卡。</p> |

{% 提示样式 = "info" %}
此功能仅在高级版本中启用。
{% endhint %}
