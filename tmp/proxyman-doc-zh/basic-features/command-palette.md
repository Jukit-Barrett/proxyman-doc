# 命令调色板

# #1.什么事？

引入可帮助开发人员的全局命令调色板:

* 快速搜索调试工具
* 切换调试工具
* 打开iOS，模拟器，Android和模拟器的新指南
* 固定你最常用的动作

{% 提示样式 = "成功" %}
使用 ⌘ ⇧p打开命令面板窗口
{% endhint %}

{% 提示样式 = "警告" %}
命令调色板在Proxyman macOS 5.26.0或更高版本上可用
{% endhint %}
