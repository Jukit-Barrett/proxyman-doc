# Swagger OpenAPI

# #1.OpenAPI

从Proxyman v5.14.0或更高版本，Proxyman可以将Proxyman请求/响应导出为OpenAPI YAML格式或HTML。

* 支持YAML格式
* 支持html-swagger OpenAPI风格
* 内置在Proxyman中，不需要任何外部依赖。

# #2.如何使用

1.使用Proxyman捕获HTTPS请求遵循以下安装指南:

* [iOS设备 (iPhone，iPad)](https://docs.proxyman.com /调试设备/ios设备)
* [iOS模拟器](https://docs.proxyman.com /调试设备/ios模拟器)
* [Android设备](https://docs.proxyman.com /调试-设备/android-设备 # id-1.-android-设置-指南)
* [Android模拟器](https://docs.proxyman.com /调试设备/android设备/自动脚本为android模拟器)
* [macOS (Web浏览器)](https://docs.proxyman.com /调试设备/macos)

验证Proxyman可以捕获和解密您的HTTPS请求/响应

<figure><img src = "images/CleanShot_2025-01-06_at_2_.47.34_2x_0f271652.jpg" alt = ""><figcaption><p> 捕获HTTPS请求/响应 </p></figcaption></figure>

2.在主表上选择您的请求-> 右键单击-> 导出-> OpenAPI -> 选择OpenAPI YAML或OpenAPI HTML
3.完成: 白色 \_复选 \_标记:
