# 本地化

### 1.什么事？

Proxyman支持本地化，这会影响应用程序的所有UI。以下列表是当前的langauges:

* 英语 (默认)
* 中文-简体

### 2.如何覆盖应用程序语言？

默认情况下，Proxyman将尊重当前的系统语言。要覆盖它，请:

* 系统首选项-> 语言和地区-> 选择应用程序选项卡-> 添加Proxyman应用程序-> 单击按钮-> 选择Proxyman应用程序和英语。

### 3.提高本地化？

如果您的语言翻译不佳，请帮助我们 < https://github.com/ProxymanApp/ proxyman-localization>
