# 自定义标题列

### 1.什么事？

自定义标题列功能允许您自定义在主表视图上显示的列。它类似于 [Google Chrome开发工具] 中的自定义标题列 ( https://developers.google.com/web/tools/ chrome-devtools/network/reference # Custom-columns)

### 2.效益

* 从请求/响应定义一个头，并显示在表上。
* ** 如果每个请求/响应具有相同的URL，但标头不同，则更容易区分 **
* 支持调整大小/排序/重新排序列
* 记住以前的状态，并为将来的会话恢复它。

### 3.如何使用它？

您可以打开 ** 工具菜单 ** -> ** 自定义标题列 **
