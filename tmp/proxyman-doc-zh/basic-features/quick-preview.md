# 快速预览

# #1.什么事？

快速视图可以帮助您在应用程序的任何位置快速预览所选内容。

* 美化内联JSON
* 解码Base64字符串
* 将key \ & value显示为表格
* 解码JWT令牌

# #2.如何使用它？

1.选择应用程序上的任何文本，例如Body选项卡，Raw选项卡等
2.右键单击以显示菜单上下文
3.选择文本选择视图作为模式
