# 断点模板

# #1.什么事？

从Proxyman 3.1.0，您可以为请求和响应创建断点模板，以允许开发人员重用它。

* 为请求创建新模板: HTTP方法，URL，标头
* 响应模板: 状态代码，标题
* 使用 [断点工具](https://docs.proxyman.com/breakpoint#1.- whats-it) 时启动生产力。

![请求/响应的断点模板](images/153364144-b1a8378e-dff8-412b-a3b6-4bfad44f9d61_c2ce95ad.png)

# #2.如何创建新模板？

### 创建一个新的

1.工具菜单-> 断点模板
2. CMD N或SHIFT CMD N为请求/响应创建模板
3.查找您的请求/响应模板

### 使用现有的

1.创建断点规则-> 确保您的请求/响应-> 确保它击中断点
2.点击原始标签-> 模板按钮-> 请求/响应-> 创建新的请求/响应
3.完成

# #3.如何使用它？

1.当请求/响应命中断点-> 单击原始选项卡
2.模板-> 请求/响应-> 选择您的模板
3.它将用模板替换当前数据。
