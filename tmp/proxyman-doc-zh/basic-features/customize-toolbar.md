# 自定义工具栏

# #1.什么事？

从Proxyman 2.32.0 +，您可以轻松自定义工具栏以满足您的需求。

您可以使用各种工具按钮设计工具栏:

* 代理覆盖
* 阻止列表
* 允许列表
* 断点
* 本地地图
* 地图远程
* 反向代理
* 网络条件
* 脚本编写

这些工具按钮将在特定功能上切换开/关。

![自定义您的工具栏] (图像/Screen_Shot_2021-08-28_at_17_15_16_4576fbe0.png)

# #2.如何使用它？

您可以简单地打开自定义工具栏调色板:

* 右键单击工具栏-> 自定义工具栏
* 工具菜单-> 自定义工具栏

{% 提示样式 = "info" %}
自定义工具栏仅适用于macOS 11 +
{% endhint %}
