# 导入/导出设置

# #1.导入设置

从Proxyman 2.26.0，开发人员可以从使用 ** Proxyman ** 或 ** Charles Proxy ** 的其他同事导入工具设置。

要导入设置，请转到工具-> 导入设置-> 从Proxyman/Charles Proxy -> 选择要导入的工具

![](图片/Screen_Shot_2021-05-17_at_09_11_44_f885dd03.png)

以下列表将介绍您可以导入的工具:

### 代理

* SSL代理列表
* 阻止列表
* 白名单
* 本地地图
* 地图远程
* 断点
* 网络条件
* 脚本编写

### Charles代理

* SSL代理列表
* 阻止列表
* 白名单
* 本地地图
* 地图远程
* 断点

# #2.导出设置

您还可以导出工具设置，然后共享给您的同事。

1.打开要导出的工具
2.选择规则
3.右键单击-> 导出设置

![](图片/Screen_Shot_2021-05-17_at_09_16_41_79633f63.png)

# #3.打开Charles代理日志文件 (\ *.chls)
