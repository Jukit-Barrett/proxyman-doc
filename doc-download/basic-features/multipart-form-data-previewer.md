# Multipart Form-Data Previewer

## 1. What's it?

From Proxyman 2.27.0+, Proxyman displays a multipart/form-data in a nice table view.

![multipart/form-data](https://1856518738-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-LlPt_6BePnJ3oK3saP1%2F-MacjaS9sQSw-8XELMKJ%2F-MackOx1UhMiBd_KrVXq%2F119465504-ea9a5580-bd6d-11eb-9d4d-3725a37f09ec.png?alt=media\&token=a6f2c849-ad11-4f37-aba3-76f723418973)

### 2. Benefit

* Display each separate part of the multipart/form-data body
* Able to export the data
