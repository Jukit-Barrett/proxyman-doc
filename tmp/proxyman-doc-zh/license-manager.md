# 许可证管理器

License Manager是管理许可证的一种更简单的方法，它允许您:

* 显示许可证状态 (购买日期，到期日期，Mac/iOS座位数)
* 查看由您的许可证密钥激活的设备列表
* 续订/延长您的许可证密钥
* 删除Mac或iOS设备

![许可证管理器](images/Proxyman_License_Manager_2c42241b.png)

# # 如何使用它？

1.打开 [许可证管理](https:// proxyman.io/License-manager/access-link)
2.输入您的电子邮件，您购买许可证密钥。
3.您将从您的邮箱收到 “访问链接”。
4.打开它以访问许可证管理器。
